## Modele Makita (uwaga: DHP490 nie istnieje!)

Prawdopodobnie chodzi o **DHP486** (pełnowymiarowa) lub **DHP487** (kompaktowa).

### Makita DHP486 (18V LXT Brushless)

| Parametr | Wartość |
|----------|---------|
| Silnik | Brushless (bezszczotkowy) |
| Moment obrotowy | 130 Nm |
| Obroty | 0–500 / 0–2100 RPM |
| Udary | 0–31 500 BPM |
| Uchwyt | 13 mm metalowy, bezkluczykowy |
| Bateria | 18V LXT (kompatybilna z całym ekosystemem Makita 18V) |
| Waga | ~2,4 kg (bez baterii) |
| Cena (samo narzędzie) | ~350–550 PLN |
| Cena (z 2× 5.0Ah + ładowarka) | ~700–1100 PLN |
| Dla kogo | Profesjonaliści, ciężkie DIY, wiercenie w betonie/murze |

### Makita DHP487 (18V LXT Brushless Sub-Compact)

| Parametr | Wartość |
|----------|---------|
| Silnik | Brushless |
| Moment obrotowy | 40 Nm |
| Obroty | 0–500 / 0–1700 RPM |
| Uchwyt | 13 mm bezkluczykowy |
| Waga | ~1,5 kg (bez baterii) |
| Cena (samo narzędzie) | ~300–450 PLN |
| Dla kogo | Lekkie prace, montaż mebli, wiercenie w drewnie/metalu |

### Porównanie DHP486 vs DHP487

| Cecha | DHP486 | DHP487 |
|-------|--------|--------|
| Moment | 130 Nm | 40 Nm |
| Rozmiar | Pełny | Kompaktowy |
| Waga | 2,4 kg | 1,5 kg |
| Beton/mur | ✅ tak | ⚠️ lekko |
| Cena | ~450 PLN | ~350 PLN |

**Zasada:** Beton/mur → DHP486. Drewno/metal/montaż → DHP487 wystarczy.

## Alternatywy z AliExpress

### Popularne marki
WOSAI, ONEVAN, HILDA, PROSTORMER, DEKO, DRILLPRO

### Typowe parametry (reklamowane vs realne)

| Parametr | Reklamowane | Realne |
|----------|-------------|--------|
| Moment obrotowy | 80–200 Nm | **30–60 Nm** |
| Silnik "brushless" | ✅ | Często tak, ale gorszej jakości |
| Uchwyt | 13 mm | Plastikowy, luźniejszy |
| Bateria | "6.0 Ah" | **Realne 2.0–3.0 Ah** |
| Cena (samo narzędzie) | — | ~80–250 PLN |
| Cena (z 2 bateriami) | — | ~200–500 PLN |

### Kompatybilność z Makita 18V
- Większość używa tego samego interfejsu baterii co Makita 18V LXT
- Działają z oryginalnymi bateriami Makita
- Działają z tanimi bateriami AliExpress (ale te mają zawyżoną pojemność i słabsze zabezpieczenia BMS)

### Jakość — co mówią testy

| Aspekt | Makita | AliExpress |
|--------|--------|------------|
| Rzeczywisty moment | 100% deklarowanego | 30–60% deklarowanego |
| Jakość wykonania | Metal, precyzyjne tolerancje | Więcej plastiku, luzy |
| Łożyska | Wysokiej jakości | Tańsze, głośniejsze |
| Bicie uchwytu | Minimalne | Wyraźne |
| Mechanizm udarowy | Trwały, mocny | Słabszy, mniej trwały |
| Żywotność | Lata codziennego użytku | 6–24 miesiące |
| Gwarancja | 2–3 lata (w PL) | Brak realnej gwarancji |

## Rekomendacja

**Jeśli masz już baterie Makita 18V:** Kup DHP486 (bare tool) — używasz istniejące baterie, oszczędzasz, narzędzie na lata.

**Jeśli nie masz ekosystemu Makita:**
- Poważne DIY → Makita DHP486 z zestawem (baterie + ładowarka) — inwestycja w ekosystem
- Okazjonalne użycie (raz na miesiąc) → AliExpress ONEVAN/WOSAI brushless za ~200 PLN jest OK
- Bardzo rzadkie użycie → najtańsza opcja AliExpress ~100 PLN, ale nie oczekuj cudów

**AliExpress warto jeśli:** potrzebujesz wiertarki na 2–3 razy w roku, nie masz baterii Makita, budżet jest kluczowy.

**AliExpress nie warto jeśli:** regularnie wierciasz, potrzebujesz niezawodności, zależy Ci na precyzji.
